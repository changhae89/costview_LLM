# CostView

CostView is now set up as an Expo Go project using React Native.

## Current Status

- Expo app boots successfully on mobile through Expo Go
- The `src` folder has been reset to a clean starting point
- `src/App.jsx` contains a minimal starter screen for rebuilding the app from scratch

## Run The Project

From the project root:

```powershell
npm.cmd start
```

Useful scripts:

```powershell
npm.cmd run android
npm.cmd run ios
npm.cmd run web
```
